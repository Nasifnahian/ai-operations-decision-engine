# AI Operations Decision Engine

A web-based analytics tool that transforms raw operational data into actionable insights.

## Features
- Upload CSV data
- Automated KPI calculations
- Downtime and error analysis
- AI-style insight generation
- Interactive dashboard visualization

## Tech Stack
- Python
- Streamlit
- Pandas
- Plotly

## Use Case
Designed for operational teams to quickly identify performance issues and improve efficiency.

## Future Improvements
- Support for dynamic CSV formats
- Real AI integration (OpenAI)
- Advanced visual dashboards
